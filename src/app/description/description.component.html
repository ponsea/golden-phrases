当アプリケーションについて

当アプリケーションは、TOEIC対策関連の書籍のベストセラーである、「TOEIC® L&R TEST 出る単特急 金のフレーズ」（通称：「金フレ」）の学習を補助するためのものです。金フレに掲載されている各フレーズを、穴埋め問題として実際に入力しながら学習することができます。解答した問題の正答率は記録されるので、自身の実力や成長度合いを確認し、学習の励みとして頂けたらと思います。全1000フレーズは、約30分で行える50フレーズごとのセクションに分けられていますので、日々の学習の目安としてください。

引用元の書籍である「TOEIC® L&R TEST 出る単特急 金のフレーズ」では、当アプリケーションで引用させていただいた1000フレーズ以外にも、TOEIC対策の要となる以下のコンテンツが掲載されています。

  • パート1重要語
  • 部署・職業名
  • 前置詞・接続詞・接続副詞
  • 多義語
  • 定型表現

TOEIC® L&R TEST 出る単特急 金のフレーズ

2012年3月発売の前作、累計40万部のベストセラーである「新TOEIC® TEST 出る単特急 金のフレーズ」を新形式に対応させた大改訂版。著者であるTEX加藤氏は、2011・12・13年のTOEIC公開テスト全26回、全て満点。2016年5月の新形式移行後も満点を連続取得している。TEX加藤氏は全ての公式教材に加え、80回以上の受験直後に書き留めた「TEXファイル」と呼ばれるデータベースから頻出語を徹底分析しており、それに基づく出題予想の正確度から圧倒的な支持を得ている。点数アップへ直接結びつけるための、真の「TOEIC対策」として欠かすことのできない一冊。
